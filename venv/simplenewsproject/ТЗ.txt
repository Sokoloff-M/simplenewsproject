Тестовое задание. Срок выполнения - неделя.

Задача:

Написать простейший новостной сайт "simplenews".
Сайт представляет собой страницу со списком заголовков новостей. Можно зайти внутрь каждой новости. Внутри выводится текст новости, список комментариев к новости и форма добавления комментария.

Новость должна содержать время, заголовок, текст. Новость можно скрыть.
К новости можно добавить комментарий. Комментарий должен содержать, время, текст. Комментарий можно скрыть.

Дизайн не важен.



Ограничения:

Бэкенд написан на django. Управление новостями и комментариями можно сделать через стандартный django admin.

Новости выводятся стандартным способом через django templates.

Комментарии (вывод и добавление) должны работать по api. (django rest framework)

Фронтенд любой.

Очень желательно покрыть тестами.

Проект должен быть завернут в docker и запускаться одной командой.

Проект должен быть выложен на github с инструкцией как запускать проект и тесты.
